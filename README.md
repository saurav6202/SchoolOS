## 📖 About

School-OS is a complete School ERP platform designed to digitize everyday school operations.

The project provides dedicated dashboards for **Students**, **Teachers**, and **Administrators**, making it easier to manage attendance, notices, authentication, and other school activities through a clean and responsive interface.

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/saurav6202/SchoolOS.git

cd school-os
```

---

### Backend

```bash
cd backend

npm install

npm run dev
```

---

### Frontend

```bash
cd frontend

npm install

npm run dev
```

---

## ⚙ Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=4000

MONGODB_URI=

JWT_SECRET=

CLIENT_URL=http://localhost:5174
```
